**Badge** — compact label for status and metadata (site health, environment, tags).

```jsx
<Badge tone="success" dot>Online</Badge>
<Badge tone="warning" dot>Updates</Badge>
<Badge tone="danger" dot>Down</Badge>
<Badge tone="neutral">Production</Badge>
```

Tones: `neutral`, `primary`, `success`, `warning`, `danger`. Pass `dot` for the leading health dot. Keep labels to one or two words.
